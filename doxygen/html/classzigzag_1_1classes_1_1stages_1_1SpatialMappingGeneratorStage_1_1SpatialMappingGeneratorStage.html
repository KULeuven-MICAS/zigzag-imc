ZigZag bridges the gap between algorithmic DL decisions and their acceleration cost on specialized accelerators through a fast and accurate HW cost estimation.

Detailed Description

Pipeline stage that finds spatial mappings given a:

  • accelerator
  • core allocation
  • interconnection pattern on the allocated core
  • layer

The spatial mappings are found using the interconnection pattern present on the core.

The inner-most memory level served dimensions is used, as this is how the memories connect to the operational array.
